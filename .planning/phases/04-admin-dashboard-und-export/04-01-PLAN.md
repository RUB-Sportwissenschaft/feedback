--- JS: Add after submitFeedback(), before INIT DOMContentLoaded block ---

Add three functions in this order:

1. initAdmin() — async function:
   - Hide main (display none), hide confirmationScreen (display none)
   - Get adminEl = getElementById('adminDashboard'), set display:block
   - Show loading message in adminEl.innerHTML
   - Fetch GET https://api.jsonbin.io/v3/b/ + JSONBIN_BIN_ID + /latest with X-Access-Key header
   - On error: show error message in adminEl
   - Parse record = (await res.json()).record || {}
   - submissions = Object.values(record)
   - adminData = computeAdminData(submissions)
   - renderAdminDashboard(adminData, adminEl)

2. computeAdminData(submissions) — returns adminData object with these keys:
   - total: submissions.length
   - groupCounts: object, group display string -> count (e.g. "Gruppe 1": 8)
   - qAverages: object, questionId -> avg (null if no ratings for that question)
     Computed by iterating submissions, accumulating sums+counts per questionId from s.ratings
   - groupAverages: object, group -> { questionId: avg | null }
     Per-group per-question averages using same accumulation pattern
   - freitexteBySection: object, sectionId -> array of { text, group }
     Iterate submissions.freitexte, push non-empty texts; keys are 'S1','S3','S2','S4','S5','allgemein'
   - ausbilderFeedback: object, ausbilderName -> array of { text, group }
     Iterate submissions.ausbilder, push non-empty texts
   - submissions: the raw array (kept for Excel export in Plan 04-02)

3. renderAdminDashboard(adminData, adminEl) — builds HTML string, sets adminEl.innerHTML:

   Build sections in this order:

   a. Header card (class="admin-header-card"):
      - Left: h2 class="admin-header-title" text "Admin-Dashboard"
        and p class="admin-header-count" text "Gesamt: N Teilnehmer"
      - Right: button id="exportBtn" class="btn btn-primary" disabled text "Excel exportieren"
        (will be wired in Plan 04-02)

   b. Section "Teilnahme pro Gruppe" (class="admin-section"):
      - h3 class="admin-section-title"
      - div class="admin-stat-grid": one .admin-stat-card per group in adminData.groupCounts
        Sort group keys: numeric groups first ('Gruppe 1'..'Gruppe 8') then SB groups
        Each card: div.stat-group = group name, div.stat-count = count

   c. Section "Durchschnittswerte pro Frage":
      - h3 class="admin-section-title"
      - table class="admin-table": thead with Frage | Ø | Balken columns
        tbody: for each key in Object.keys(QUESTION_LABELS) order:
          td: QUESTION_LABELS[qId]
          td: avg.toFixed(1) or "—"
          td: div.avg-bar-wrap containing div.avg-bar with inline style width=(avg/5*100)+'%'
            (0% width if null)

   d. Section "Gruppenvergleich":
      - h3 class="admin-section-title"
      - table class="admin-table": thead Gruppe | Ø gesamt
        tbody: for each group in sorted groupAverages keys:
          Compute overallAvg = mean of all non-null question averages for that group
          td: group name, td: overallAvg.toFixed(2) or "—"

   e. Section "Freitextantworten":
      - h3 class="admin-section-title"
      - For each key in Object.keys(SECTION_LABELS):
        Skip if freitexteBySection[key] empty or undefined
        h4 = SECTION_LABELS[key], font-size 0.875rem, color var(--text-muted), margin 0.5rem 0
        For each entry: div.admin-freitext-card containing
          p.admin-freitext-text = entry.text
          p.admin-freitext-meta = entry.group

   f. Section "Ausbilder-Feedback":
      - h3 class="admin-section-title"
      - For each ausbilderName in Object.keys(ausbilderFeedback).sort():
        Skip if no entries
        h4 = ausbilderName (same style as above)
        For each entry: div.admin-freitext-card same structure
