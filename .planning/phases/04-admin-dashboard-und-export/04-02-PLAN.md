--- 3. Add exportToExcel(adminData) function ---

Place after renderAdminDashboard(), before the closing script tag.

The function builds a workbook with two sheets:

SHEET 1 — "Einzelantworten" (one row per submission):

Header row: ['Timestamp', 'Gruppe', 'Name',
  ...Object.keys(QUESTION_LABELS),  // 18 question IDs as column headers
  'S1 Freitext', 'S3 Freitext', 'S2 Freitext', 'S4 Freitext', 'S5 Freitext', 'Allgemeines Freitext',
  'Ausbilder Feedback'  // combined "Name: text" pairs joined by " | "
]

Data rows: For each submission in adminData.submissions:
  - timestamp: s.timestamp
  - group: s.group
  - name: s.name || ''
  - ratings: Object.keys(QUESTION_LABELS).map(q => s.ratings ? (s.ratings[q] || '') : '')
  - freitexte per section: ['S1','S3','S2','S4','S5','allgemein'].map(k => s.freitexte ? (s.freitexte[k] || '') : '')
  - ausbilder combined: s.ausbilder
      ? Object.keys(s.ausbilder).map(n => n + ': ' + s.ausbilder[n]).join(' | ')
      : ''

Build with XLSX.utils.aoa_to_sheet([headerRow, ...dataRows])

SHEET 2 — "Zusammenfassung":

Part A: Per-question averages
  Header row: ['Frage-ID', 'Frage', 'Durchschnitt (alle Gruppen)']
  One row per question in QUESTION_LABELS order:
    [qId, QUESTION_LABELS[qId], adminData.qAverages[qId] !== null ? +adminData.qAverages[qId].toFixed(2) : '']

Blank row separator: ['']

Part B: Per-group overall averages
  Header row: ['Gruppe', 'Ø gesamt', 'N Teilnehmer']
  One row per group in sorted order (numeric groups then SB groups):
    Compute overallAvg = mean of non-null values in adminData.groupAverages[g]
    [group, overallAvg !== null ? +overallAvg.toFixed(2) : '', adminData.groupCounts[g] || 0]

Build with XLSX.utils.aoa_to_sheet(allRows) where allRows = [
  partA header, ...partA data rows, blank row, partB header, ...partB data rows
]

Assemble and write:
  var wb = XLSX.utils.book_new();
  XLSX.utils.book_append_sheet(wb, sheet1, 'Einzelantworten');
  XLSX.utils.book_append_sheet(wb, sheet2, 'Zusammenfassung');
